1. Til þess að keyra appið þarf að setja upp forritasöfn, sjá lið 2.
Vefþjón er keyrður með skipuninni python main.py í rót verkefnis.
Gagnagrunnur er tilbúinn til notkunnar.

2. Forritasöfn
Það þarf að sækja þessi library til þess að geta keyrt forritið.
Til þess að fá flask, werkzeug og jinja þar að setja upp flask:
http://flask.pocoo.org/docs/installation/#installation
{
    flask
    werkzeug
    jinja2
}
Til þesss að fá orm/rest api mapperinn og háð forrita söfn (wtforms, wtf-peewee, flask_peewee) þarf að setja upp peewee:
https://github.com/coleifer/flask-peewee
{
    peewee
    wtforms
    wtf-peewee
    flask_peewee
}
Til þess að fá request http lib þarf að setja upp requests:
http://pypi.python.org/pypi/requests
{
    requests
}
Til þess að fá eyeD3 tag lib þarf að setja upp eyeD3:
http://eyed3.nicfit.net/
{
    eyeD3
}
